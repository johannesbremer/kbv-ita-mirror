RELEASE NOTES KBV-Prüfmodul für DMP Brustkrebs 4a Prüfung Version 2026.1.0 1/2026

1. Systemvoraussetzungen
- ein beliebiges Computersystem leistungsmäßig vergleichbar mit einem IBM-kompatiblen PC
  mit Pentium 500 Prozessor oder höher
- Betriebssystem, für das die Java Laufzeitumgebung verfügbar ist
- Java Laufzeitumgebung 17 oder höher
- mindestens 128 MB RAM
