Systemvoraussetzungen

  • Benötigt wird ein beliebiges Computersystem leistungsmäßig vergleichbar mit einem IBM-kompatiblen PC mit Pentium 500 MHz Prozessor oder höher.
  • Das XPM verwendet die plattformunabhängige Java Technologie. Dies ermöglicht den Einsatz dieser Applikation auf verschiedenen Betriebssystemen, für die die Java Laufzeitumgebung verfügbar ist.
  • Java Laufzeitumgebung 17 oder höher
  • Arbeitsspeicher mindestens 256 MB RAM
  • Falls unter Windows oder Linux durch die GUI ein PDF angezeigt werden soll, muss ein PDF-Viewer installiert sein.
