Systemvoraussetzungen

  • Alle Meldungen dieses XPM-Paketes sind in der Meldungsliste dokumentiert.
  • Benötigt wird ein beliebiges Computersystem leistungsmäßig vergleichbar mit einem IBM-kompatiblen PC mit Pentium 500 MHz Prozessor oder höher.
  • Das XPM verwendet die plattformunabhängige Java Technologie. Dies ermöglicht den Einsatz dieser Applikation auf verschiedenen Betriebssystemen, für die die Java Laufzeitumgebung verfügbar ist.
  • Java Laufzeitumgebung 17 oder höher
  • Arbeitsspeicher mindestens 128 MB RAM
  • Auf dem System muss für die Ausgabe von PDF-Protokollen die Schriftart Arial installiert sein, falls nicht wird eine Warnmeldung auf der Konsole ausgegeben und die Schriftart Sans Serif verwendet.

Bekannte Fehler

  • Nach Start des XPM-Prüfmodules MIT grafischer Benutzeroberfläche kann es unter Windows 98 vorkommen, dass keine Bildschirmanzeige mehr dargestellt wird (schwarzer Bildschirm). Dieser Fehler ist vermutlich auf Kompatibilitätsprobleme zwischen Windows 98 und Java Swing zurückzuführen. Eine Nutzung des XPM-Prüfmodules OHNE grafische Benutzeroberfläche ist jedoch uneingeschränkt möglich.
  • Die Verwendung von Umlauten in Verzeichnisnamen kann zum Abbruch einer Prüfung führen.
