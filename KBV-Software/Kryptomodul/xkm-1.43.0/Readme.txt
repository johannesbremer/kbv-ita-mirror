Programm XKM Kryptomodul Version 1.43.0

Zu diesem Programm gehören folgende Dateien:

- ReleaseNotes.html                 Hinweise zum aktuellen Release
- Readme.txt                        Diese Datei
- Installation.txt                  Hinweise zur Installation

Im Ordner XKM:

- StartKryptomodul.bat              Batch für Entschlüsselung im Konsolenbetrieb (Batchdatei für Windows)
- StartKryptomodul.sh               Skript für Entschlüsselung im Konsolenbetrieb (Shellskript für Unix)
- StartGUIKryptomodul.bat           Verschlüsselung mit grafischer Oberfläche (Shellskript für Windows)
- StartGUIKryptomodul.sh            Verschlüsselung mit grafischer Oberfläche (Shellskript für Unix)
- ErzeugeBenutzerschluessel.bat     Generierung eines benutzerspezifischen Schlüsselpaars (Batchdatei für Windows)
- ErzeugeBenutzerschluessel.sh      Generierung eines benutzerspezifischen Schlüsselpaars (Shellskript für Unix)
- ErzeugeGUIBenutzerschluessel.bat  GUI-Dialog zur Generierung eines benutzerspezifischen Schlüsselpaars (Batchdatei für Windows)
- ErzeugeGUIBenutzerschluessel.sh   GUI-Dialog Generierung eines benutzerspezifischen Schlüsselpaars (Shellskript für Unix)

Verzeichnisse:

- Bin                               Beinhaltet alle Java-Archive, die zur Ausführung des neuen KBV-Kryptomodul benötigt werden.
- Ausschuss                         Default-Verzeichnis für abgelehnte Dateien
- Entschluesselt                    Default-Verzeichnis für entschlüsselte Dateien
- Verschluesselt                    Default-Verzeichnis für verschlüsselte Dateien
- Quelle                            Default-Verzeichnis für eingehende Dateien zum Verschlüsseln
- Bearbeitet                        In diesem Ordner werden die im Serverbetrieb ohne Löschmodus abgearbeiteten Dateien gesammelt.
- Schluessel                        Default-Verzeichnis für generierte Schlüssel
- Listen                            Default-Verzeichnis zur Ablage von Listen und Protokollen
- System                            Nur für interne Zwecke!
- Doku                              Enthält ausgelieferte Dokumentationen
- Konfig                            Enthält Konfigurationsdateien 'config.xml' und 'schluessel.xml'
